# ![Cre8bit logo](/logo.png) cre8bit-minified
**This is a slightly minified version of [cre8bitJS](https://www.npmjs.com/package/cre8bitjs)**
## Setup
Via npm
```
$ npm install cre8bitjs-minified --save
```
```javascript
import cre8bit from 'cre8bitjs-minified';
```
